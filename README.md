# Snake Game V2
Snake Game from java using game engine(libGDX)

## Usage
- vs code
- Java
- Gradle
- game engine ([`libGDX`](https://libgdx.com/dev/))

## Requirements
- Any Java IDE
- java version 8+
- gradle (optional)

> Note: Open terminal, First type gradlew build or gradle build; then gradlew run or gradle run;
